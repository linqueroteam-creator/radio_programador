# ANOTATA — Seu bloco de notas na web

## O que é?
ANOTATA é um site de anotações pessoal (tipo Evernote), que roda direto no
navegador. Suas notas ficam salvas no próprio navegador — não precisa instalar
nada no computador.

## Como acessar
Basta abrir o link do site (GitHub Pages) que será fornecido.
Funciona em qualquer navegador (Chrome, Edge, Firefox, Safari).

No celular, depois de abrir o site, você pode "Adicionar à tela de início"
pra ele virar um aplicativo de verdade — abre sem barra do navegador,
funciona offline.

## O que dá pra fazer

### Escrita rica
- Editor com negrito, itálico, sublinhado, cores, destaque
- Cabeçalhos, listas, listas numeradas, citações, divisores
- Inserir imagens (cole com Ctrl+V ou arraste pra dentro)
- Imagens redimensionáveis clicando nelas
- Links normais e **links internos** entre notas (selecione texto → "Ligar a...")

### Comandos rápidos com `/`
Ao digitar `/` no editor, abre um menu flutuante com:
- Cabeçalho 1, 2, 3
- Lista, lista numerada, lista de tarefas
- Citação, código, divisor
- Data de hoje, data e hora
- Inserir imagem

Use as setas pra navegar, Enter pra aplicar, Esc pra fechar.

### Reescritor inteligente (sem IA)
Selecione texto → botão flutuante "Reescrever" → escolha entre 5 estilos
(geral, formal, conciso, fluente, simples). Funciona 100% offline,
direto no seu navegador.

### Lembretes de prazo no navegador
Clique no sininho 🔔 da barra lateral pra ativar lembretes. O ANOTATA
te avisa pelo navegador quando uma nota tem prazo:
- ⚠️ Vencido
- 🔥 Vence hoje
- 📅 Vence amanhã

Os avisos aparecem mesmo se a aba do ANOTATA estiver minimizada.

### Exportar em lote
Em qualquer caderno: passe o mouse e clique em "..." → **Exportar caderno**.
Ou clique em "Exportar tudo" no topo da tela inicial.

Você escolhe:
- Todas as notas ou só favoritas
- Formato Markdown (.md), Texto puro (.txt) ou os dois
- Recebe um pacote ZIP com tudo + um LEIA-ME explicando o conteúdo

### Coleções automáticas
A barra lateral mostra coleções inteligentes que se preenchem sozinhas:
- Recentes, Pendentes, Sem revisão há tempo, Vencidas, Sem tags...
- Aparecem só quando têm conteúdo, ordenadas por relevância

### Conexões entre notas
Cada nota mostra outras notas relacionadas com motivo da conexão.
Você pode aceitar a sugestão, ignorar, ou criar uma conexão manual.

### Mapa visual
No editor, clique no ícone de mapa pra ver as conexões da nota como um grafo.

### Central de comandos (Ctrl+K)
Aperte **Ctrl+K** (ou ⌘+K no Mac) pra abrir uma busca universal:
buscar nota, criar com modelo, navegar, exportar, ações na nota atual.

## Atalhos de teclado
- `Ctrl+K` — Central de comandos
- `Ctrl+N` — Nova nota
- `Ctrl+Shift+N` — Nova com modelo
- `Ctrl+D` — Duplicar nota atual
- `Ctrl+Shift+F` — Favoritar
- `Ctrl+Shift+P` — Fixar no topo
- `Esc` — Fechar modal/gaveta

## Mobile (celular e tablet)
- Layout responsivo: tudo cabe e funciona com toque
- Gavetas laterais: arraste do canto esquerdo da tela pra abrir o menu
- Cards de nota: arraste pra esquerda pra arquivar, pra direita pra favoritar
- Toolbar do editor rola horizontalmente
- Áreas de toque grandes (mínimo 44×44px) — confortável de usar com o dedo
- Instale como app pelo "Adicionar à tela de início"

## Sobre a IA
A porta está preparada. Por enquanto, ANOTATA é 100% local — toda a
inteligência é regra determinística no seu navegador. Só o corretor
ortográfico usa internet (LanguageTool, gratuito).

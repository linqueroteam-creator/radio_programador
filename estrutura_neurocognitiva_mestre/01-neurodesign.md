# 🧠 01. NeuroDesign — Pesquisa Profunda

## Definição

NeuroDesign é o estudo de como decisões visuais influenciam atenção, percepção, emoção, memória, esforço mental e comportamento do usuário.

No Note Pulse, essa área serve para criar interfaces que parecem **inteligentes, vivas, claras e bonitas** sem causar poluição visual.

---

## 1. Percepção Visual
**Pesquisar:** como o cérebro interpreta formas, percepção de bordas, contraste, figura e fundo, agrupamento, padrões, leitura rápida de elementos, percepção pré-consciente, periférica, central, profundidade, movimento, brilho, espaço vazio.
**Subtópicos:** Gestalt visual, figura-fundo, proximidade, similaridade, continuidade, fechamento, simetria, pregnância, affordance visual, saliência visual.
**Aplicação no Note Pulse:** entender rapidamente o que é núcleo, raiz, pulso, região e conexão. Evitar que o mapa vire ruído visual. Ajudar o usuário a "ler" o sistema sem explicação.

## 2. Contraste
**Pesquisar:** contraste de cor, luminosidade, tamanho, movimento, espessura, ativo vs inativo, foco vs fundo, contraste emocional, semântico, funcional.
**Subtópicos:** alto, baixo, acessível, intencional, para foco, para hierarquia, para alerta, para leitura longa.
**Aplicação no Note Pulse:** pulso ativo destaca, pulso morto apaga sem sumir, raiz principal tem presença, fundo não compete com conteúdo.

## 3. Foco Ocular
**Pesquisar:** padrões de escaneamento, ponto focal, atenção periférica, captura involuntária, eye tracking, tempo de fixação, desvio.
**Subtópicos:** scan path, F-pattern, Z-pattern, focal point, visual anchor, gaze direction, attention capture.
**Aplicação no Note Pulse:** usuário entende "onde estou pensando agora", núcleo não compete com tudo, mapa conduz olhar naturalmente, editor mantém foco no texto.

## 4. Hierarquia Visual
**Pesquisar:** ordem de leitura, escala, peso visual, prioridade, agrupamento por importância, hierarquia tipográfica, cromática, dinâmica, por movimento, por proximidade.
**Subtópicos:** foco primário, secundário, suporte, latentes, fundo.
**Aplicação no Note Pulse:** núcleo mestre maior que subnúcleos, subnúcleo maior que pulsos, pulsos ativos mais fortes que antigos, sinapses fortes mais visíveis.

## 5. Leitura Cognitiva
**Pesquisar:** como cérebro entende interfaces, interpretação de símbolos, leitura sem esforço, clareza, previsibilidade, padrões, significado visual, interpretação rápida, interface autoexplicativa.
**Subtópicos:** cognitive readability, visual comprehension, cognitive affordance, semantic clarity, mental model, recognition over recall.
**Aplicação no Note Pulse:** pulso parece anotação (não bolinha abstrata), mapa é vivo mas compreensível, estados são visuais sem depender de texto.

## 6. Fadiga Visual
**Pesquisar:** excesso de estímulos, poluição, brilho excessivo, animação excessiva, saturação, leitura longa, sobrecarga, ruído.
**Subtópicos:** visual fatigue, cognitive overload, attention fatigue, animation overload, low-energy mode.
**Aplicação no Note Pulse:** movimento sutil, glow não constante, mapa respira, modo escrita acalma o sistema.

## 7. Peso Visual
**Pesquisar:** elementos que parecem mais importantes — influência de tamanho, cor, posição, brilho, densidade, espaço ao redor, formas circulares vs orgânicas, equilíbrio visual.
**Subtópicos:** visual weight, balance, emphasis, dominance, focal mass, negative space.
**Aplicação no Note Pulse:** núcleo central tem peso simbólico maior, subnúcleos peso intermediário, pulsos peso variável por energia, conexões não pesam mais que pensamentos.

## 8. NeuroEstética
**Pesquisar:** beleza percebida pelo cérebro, harmonia, proporção, fluidez, simetria, formas orgânicas vs geométricas, prazer visual, premium, sensação de inteligência visual.
**Subtópicos:** neuroaesthetics, processing fluency, beauty + usability, harmony, elegance.
**Aplicação no Note Pulse:** mapa bonito sem virar arte inútil, visual gera vontade de usar, beleza apoia clareza.

## 9. Movimento Visual
**Pesquisar:** movimento que guia atenção, orgânico, respiração, energia, microanimações, transições, SVG/WebGL/CSS/partículas.
**Subtópicos:** motion design, organic animation, breathing animation, pulse, particle flow, motion hierarchy, reduced motion, easing.
**Aplicação no Note Pulse:** núcleo respira, raízes tipo água-viva, energia nos fios neurais, pulsos com micropulsação, conexões recentes com fluxo sutil.

## 10. Cor Emocional
**Pesquisar:** psicologia, emoção, foco, energia, confiança, urgência, calma, memória, paletas cognitivas, cores semânticas.
**Subtópicos:** color psychology, semantic color, color hierarchy, accessibility, temperature, saturation, hue, luminance.
**Aplicação no Note Pulse:** roxo/lilás = identidade cognitiva, verde-lima = atividade/vida, âmbar = urgência/retomada, cinza = pulso morto. Cores dos cérebros têm função, não decoração.

## 11. Espaço em Branco
**Pesquisar:** respiro, silêncio visual, margem cognitiva, densidade, agrupamento por espaço, foco por isolamento, leitura confortável, estética premium.
**Subtópicos:** whitespace, breathing room, premium spacing.
**Aplicação no Note Pulse:** mapa não fica lotado, editor com espaço de escrita, painel inteligente com poucos elementos, zoom revela complexidade progressivamente.

## 12. Tipografia Cognitiva
**Pesquisar:** legibilidade, leitura longa, tamanho/altura/largura, peso, contraste tipográfico, fontes para SaaS/anotação/painel cognitivo.
**Subtópicos:** readability, line-height, type scale, variable fonts, editorial interface.
**Aplicação no Note Pulse:** Pulso confortável como Evernote/Notion, título com presença, conteúdo agradável para leitura longa, painel direito escaneável.

## 13. Saliência Visual
**Pesquisar:** o que chama atenção primeiro — movimento, cor, tamanho, brilho, novidade, competição por atenção.
**Subtópicos:** visual salience, bottom-up vs top-down, novelty detection, signal vs noise.
**Aplicação no Note Pulse:** só um elemento domina por vez, energia alta atrai sem gritar, IA destaca sem roubar foco, notificação rara e útil.

## 14. Previsibilidade Visual
**Pesquisar:** padrões familiares, convenções, previsibilidade estrutural, confiança, consistência, comportamento esperado, quebra controlada de expectativa.
**Subtópicos:** predictive processing, mental model, controlled novelty.
**Aplicação no Note Pulse:** editor familiar, mapa pode inovar, botões se comportam como botões, novidade visual fica no mapa.

## 15. Novidade Visual Controlada
**Pesquisar:** interesse, surpresa positiva, novidade sem confusão, diferenciação, micro surpresas, estética proprietária.
**Subtópicos:** novelty effect, positive prediction error, signature interaction, brand behavior.
**Aplicação no Note Pulse:** núcleo vivo como assinatura, raízes orgânicas como diferencial, pulsos com energia como linguagem própria. Sem espetáculo.

## 16. Acessibilidade Visual
**Pesquisar:** contraste acessível, daltonismo, legibilidade, reduced motion, tamanhos mínimos, foco visível, navegação teclado, acessibilidade cognitiva.
**Subtópicos:** WCAG, color blindness, focus state, inclusive design.
**Aplicação no Note Pulse:** cor não é única forma de comunicar estado, animações têm modo reduzido, mapa compreensível sem glow, editor confortável uso longo.

## 17. Densidade Visual
**Pesquisar:** quantidade de elementos por tela, complexidade, camadas, progressive disclosure, zoom levels, revelação gradual.
**Subtópicos:** information density, focus mode, complexity management.
**Aplicação no Note Pulse:** zoom out = estrutura, zoom in = detalhe, pulsos por camadas, mapa não mostra tudo de uma vez.

## 18. Design Emocional Visual
**Pesquisar:** vínculo emocional, segurança, presença, confiança estética, sistema acolhedor, produto memorável.
**Subtópicos:** emotional design, affective interface, trust signals, calm technology, perceived intelligence.
**Aplicação no Note Pulse:** sistema vivo, não ameaçador. IA assistiva, não invasiva. Visual acolhe em dias de baixa energia. Transmite confiança.

## 19. Materialidade Digital
**Pesquisar:** sensação tátil visual, glassmorphism, soft UI, profundidade leve, superfícies, camadas translúcidas, sombras naturais.
**Subtópicos:** digital materiality, soft UI, glassmorphism, depth cues, surface design.
**Aplicação no Note Pulse:** painéis com profundidade suave, mapa com camadas vivas, cards com presença leve. Evitar aparência chapada.

## 20. Identidade Visual Cognitiva
**Pesquisar:** assinatura visual, linguagem proprietária, consistência de marca, reconhecimento, comportamento da marca, design system cognitivo.
**Subtópicos:** visual identity, design tokens, motion identity, signature UI.
**Aplicação no Note Pulse:** núcleo vivo como símbolo funcional, roxo/lilás base cognitiva, energia visual como linguagem proprietária, mapa como diferencial memorável.

---

## O que evitar

- Tratar NeuroDesign como “interface bonita”
- Decoração sem função cognitiva
- Estética que compete com clareza

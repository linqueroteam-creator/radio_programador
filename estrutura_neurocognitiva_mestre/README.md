# 🧠 Estrutura Neurocognitiva Mestre — Note Pulse

> Base de inteligência cognitiva que governa todas as decisões de design, UI/UX, comportamento e arquitetura do **Note Pulse**.

---

## O que é esse documento

Este conjunto de arquivos é a **bússola conceitual** do projeto Note Pulse.
Toda decisão visual, de fluxo, de animação, de cor, de movimento, de sensação, de comportamento da IA, de organização espacial e de identidade do produto deve consultar este material **antes de ser tomada**.

Não é teoria solta. É a alma do sistema traduzida em camadas neurocognitivas que precisam estar presentes em cada elemento que aparece na tela.

## Como consultar

Antes de implementar qualquer feature, mudança visual ou ajuste comportamental:

1. **Identificar** quais camadas neurocognitivas a feature toca (ex: hover de uma nota toca NeuroInteração + NeuroFeedback Cognitivo + NeuroEnergia)
2. **Ler** os arquivos relevantes
3. **Aplicar** as decisões respeitando o que cada camada governa
4. **Evitar** o que cada arquivo lista como "o que evitar"

---

## Filosofia central

> O Note Pulse não é um app de notas com mapa.
> É uma **extensão cognitiva viva da mente humana**.

Cada decisão técnica precisa amplificar **clareza intelectual**, **continuidade mental**, **percepção de inteligência viva** e **conforto cognitivo**.

Nada é decoração. Tudo é função neurocognitiva.

---

## O que NUNCA fazer

- Tratar qualquer área neuro como teoria solta
- Implementar feature sem checar quais camadas ela ativa
- Substituir clareza por espetáculo
- Substituir presença viva por movimento frenético
- Substituir inteligência invisível por interface poluída
- Misturar com projetos antigos
- Quebrar a continuidade conceitual entre camadas

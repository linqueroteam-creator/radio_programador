
============================================================
  RADIO PROGRAMADOR v6.0
  Ferramenta de Auditoria Wi-Fi — Windows 11
============================================================


╔══════════════════════════════════════════════════════════╗
║  COMO GERAR O EXECUTAVEL (.exe) — 3 PASSOS             ║
╚══════════════════════════════════════════════════════════╝

  PASSO 1:  Instale o Python (so precisa fazer 1 vez)
            Baixe em: https://www.python.org/downloads/
            IMPORTANTE: Marque a caixa "Add Python to PATH"!

  PASSO 2:  De dois cliques em "GERAR_EXE.bat"
            Espere 2 a 5 minutos (ele faz tudo sozinho)

  PASSO 3:  Pronto! Seu arquivo esta em:
            PRONTO\RadioProgramador.exe

  Depois disso, voce so precisa do RadioProgramador.exe!
  Copie pra onde quiser. Pode deletar todo o resto.


╔══════════════════════════════════════════════════════════╗
║  O QUE ACONTECE QUANDO VOCE ABRE O .EXE                ║
╚══════════════════════════════════════════════════════════╝

  - O Windows vai mostrar uma tela escura perguntando:
    "Deseja permitir que este app faca alteracoes?"
  - Clique SIM (isso da permissao de Administrador)
  - O programa abre normalmente com todas as funcoes


╔══════════════════════════════════════════════════════════╗
║  REQUISITO EXTRA (para funcoes avancadas)               ║
╚══════════════════════════════════════════════════════════╝

  Para usar Captura de Handshake e Desconectar Clientes:
  - Instale o Npcap: https://npcap.com
  - Na instalacao, marque "WinPcap API compatibility"
  - Isso e gratuito e seguro


╔══════════════════════════════════════════════════════════╗
║  O QUE O PROGRAMA FAZ (9 funcoes)                       ║
╚══════════════════════════════════════════════════════════╝

  1. REDES         Busca redes Wi-Fi visiveis ao redor
  2. AUDITORIA     Testa senhas WPA2 com lista de senhas
  3. HANDSHAKE     Captura dados de conexao da rede
  4. DISPOSITIVOS  Mostra aparelhos conectados na rede
  5. VELOCIDADE    Mede velocidade da internet
  6. GERADOR       Cria senhas aleatorias fortes
  7. INFO REDE     Mostra dados completos da conexao
  8. WORDLIST      Gerencia lista de senhas para teste
  9. LOG           Registro de tudo que foi feito


╔══════════════════════════════════════════════════════════╗
║  ARQUIVOS DESTA PASTA (so precisa pra gerar o .exe)     ║
╚══════════════════════════════════════════════════════════╝

  radio_programador.py   Codigo do programa (nao mexer)
  instalador.py          Instalador auxiliar
  GERAR_EXE.bat          Gera o .exe (duplo-clique!)
  versao.rc              Informacoes do executavel
  RadioProgramador.spec  Configuracao de build
  LEIA_PRIMEIRO.txt      Este arquivo

  Depois de gerar o .exe, voce pode deletar tudo.
  So precisa do RadioProgramador.exe que fica em PRONTO\


╔══════════════════════════════════════════════════════════╗
║  PROBLEMAS COMUNS                                        ║
╚══════════════════════════════════════════════════════════╝

  "Python nao encontrado"
  → Reinstale o Python e marque "Add Python to PATH"

  "Antivirus bloqueou o .exe"
  → Isso e normal com programas gerados por PyInstaller
  → Adicione uma excecao no antivirus para o arquivo

  "Npcap nao instalado"
  → Baixe em https://npcap.com (gratuito)
  → Marque "WinPcap API compatibility" na instalacao
